tail.writer Features

Markdown Actions

  • Headers (as Dropdown or Dialog)
  • Bold
  • Italic
  • Strikethrough
  • Underline (against [GF] Markdown)
  • Link (as Button, Dropdown or Dialog)
  • Image Link (as Button, Dropdown or Dialog)
  • Indent
  • Outdent
  • Inline Code
  • Block Code
  • Horizontal Rule
  • Quote
  • (Un-) Ordered List
  • (Un-) Checked List
  • Table (as Dropdown or Dialog)
  • Preview (requires marked)
  • Separators

Editor Features

  • Still Alpha
  • Beautiful Octicons (Check it out)
  • WriteFlow supportive Functions
  • Resizable Editor
  • Knowing Tooltips
  • Counting Statusbar
  • Translatable Interface
  • Configurable System
  • Extendable Environment
  • jQuery and MooTools Bindings
  • Available for Internet Explorer 9+
  • Free and Open Source (MIT Licensed)

Editor Themes

  • Minimalistic Default Theme (Take a Look)
  • Beautiful Dark Theme (Take a Look)
  • Awesome GitHub Theme (Take a Look)

The GitHub Flavored Markdown is the only supported Markup!

But the BBCode and Textile Markups are available soon!


tail.writer Support

The tail.writer Markup Editor is still in Development and requires your help to fix even the smallest errors, bugs and inconsistencies! So please report each issue and request all of your ideas and suggestions on our official GitHub Repository.
